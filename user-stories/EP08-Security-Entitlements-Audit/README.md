# Epic EP08 — Security / Entitlements / Audit

Scope: FR5, security, audit, and data governance sections.

Goal: Provide authentication, authorization, entitlements, and immutable audit logging so that access to market data is controlled and traceable.

User stories:

- US08-01 — Service-to-service mTLS
- US08-02 — OAuth2/JWT authentication for control plane
- US08-03 — Gateway authentication via API keys or tokens
- US08-04 — RBAC and attribute-based entitlements model
- US08-05 — Entitlement enforcement in subscription and fanout
- US08-06 — Immutable audit log for admin and replay actions
- US08-07 — Data classification and governance tags
