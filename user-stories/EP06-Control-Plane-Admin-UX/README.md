# Epic EP06 — Control Plane & Admin UX

Scope: FR8, parts of FR5, control APIs, and admin UI.

Goal: Provide a Spring Boot control plane with admin APIs and UI for configuring tenants, instruments, feeds, subscription policies, feature flags, and schema registry.

User stories:

- US06-01 — Control plane service skeleton
- US06-02 — Tenants and environments management
- US06-03 — Instruments master and symbol mapping CRUD
- US06-04 — Feed configuration management
- US06-05 — Subscription policies and rate limits
- US06-06 — Feature flags and canary configurations
- US06-07 — Schema registry admin UI
